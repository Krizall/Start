Зайти - cd "имя"

Выйти - cd ..

Посмотреть - ls

Создать папку - mkdir название

Удалить директорию(папку) - rmdir название 

Создать файл - touch имя 

Редактировать файл - vim название с расширением 

Режим редактирование - insert (нажать кнопку I)

Сохранить изменения - нажать кнопку escape

Сохранить - нажать / потом вписать :w  

Выйти из режима редактирования - / потом :q

Выйти и сохранить сразу - / потом :x

Удалить файл - rm имя 

Очистить терминал визуально - clear

Посмотреть гит статус - (git status)

Отслеживать папку - git init (сначала надо в нее зайти)

Отслеживать что-то отдельно - git add название 

Отслеживать все компоненты в папке - git add .
